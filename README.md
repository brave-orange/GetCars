## 功能
多线程抓取汽车之家的品牌信息和各品牌汽车型号信息并且存入数据库
## 依赖
使用前请安装好pyquery和pymysql：
* `$ pip install pymysql`
* `$ pip install pyquery`
## 运行环境
* 开发语言：python3
* 系统： Windows/Linux
## 使用说明
1. 下载脚本：
    `$ git clone https://github.com/brave-orange/GetCars`
2. 导入数据表：`car.sql`、`brand.sql`
3. 运行car.py
